# freight-app

Aplicação que gerencia a frota de entrega para um e-commerce com alcance nacional (projeto TreinaDev 9).

#### Versão do Ruby/Rails
ruby 3.0.2 <br>
Rails 7.0.4<br>
<br>

#### Gems necessárias
- rspec-rails 5.1.2
- capybara 3.37.1
- devise 4.8.1

Para o CSS: <br>
- bootstrap 5.2.1
- importmap-rails
<br>

#### Populando o banco de dados
No terminal, use `rails db:seed`
